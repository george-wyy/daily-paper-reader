## 摘要
在记忆维持期间进行的眼跳会优先处理眼跳目标的记忆，但尚不清楚这种益处是特定于某个位置，还是会扩展到被记忆的物体上。在三个实验中，我们研究了眼跳选择是否会扩散到同一物体内的其他位置。在实验1中，我们要求观察者记住三个定向的Gabor斑，这些Gabor斑要么呈现在由轮廓定义的物体内，要么没有物体结构。随后的运动线索提示观察者将眼睛移动到指示的位置。然后，我们探测了与眼跳目标等距位置（位于同一物体或不同物体中）的刺激记忆。结果显示，对于与眼跳目标一致位置的刺激，记忆效果最好；而对于呈现在与眼跳目标相同或不同物体中的其他刺激，记忆效果一致较弱。在实验2中，我们通过向刺激添加物体特征创建了更复杂的物体。同样，与眼跳目标位置一致的刺激表现最好，而在不一致的试验中，无论刺激是在与眼跳目标相同还是不同的物体中，记忆表现都较差且相似。在实验3中，我们通过变化检测任务测试了眼跳选择是否存在并在物体内传播。同样，记忆表现（即变化检测）在眼跳目标位置最好。然而，这种记忆益处也扩散到了同一物体内的其他位置。我们的结果表明，视觉工作记忆中的眼跳选择主要是基于空间的，但也可以向眼跳所指向的物体内的位置扩散。

## Abstract
Saccades made during memory maintenance prioritize memory for the saccade target, but it is unclear if this benefit is specific to a location or extends across memorized objects. In three experiments, we examined whether saccadic selection spreads to other locations within the same object. In Experiment 1, we asked observers to remember three oriented Gabors presented either within contour-defined objects or without object structure. A subsequent movement cue prompted observers to move their eyes to the indicated location. We then probed memory for stimuli at locations equidistant from the saccade target, in either the same or a different object. Memory was best for stimuli at locations congruent with the saccade target, and consistently weaker for other stimuli presented in the same or a different object than the saccade target. In Experiment 2, we created more complex objects by adding object features to the stimulus. Again, performance was best for stimuli congruent with the saccade target location, whereas memory in incongruent trials was worse and similar for stimuli in the same and different object as the saccade target. In Experiment 3, we tested whether saccadic selection is present and propagates within the object in a change detection task. Again, memory performance (i.e., change detection) was best at the saccade target location. However, this memory benefit also spread to other locations within the same object. Our results imply that saccadic selection in visual working memory is primarily space-based but can also spread towards locations within the object where a saccade was directed.
