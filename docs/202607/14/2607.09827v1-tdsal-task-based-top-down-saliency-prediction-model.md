## 摘要
视觉显著性旨在预测图像中最可能吸引人类视觉注意的区域。虽然大多数显著性模型假设自由观看条件，但人类注意力往往受明确任务目标的影响。在这项工作中，我们通过提出一个将视觉注意力条件化于自然语言任务描述的模型，来解决任务驱动的显著性预测问题。该模型生成依赖任务的显著性图，反映在不同观看意图下注意力的转移。通过定量和定性分析，我们表明纳入明确的任务语义能够更真实地建模目标导向的视觉注意力。

## Abstract
Visual saliency aims to predict the regions of an image most likely to attract human visual attention. While most saliency models assume free-viewing conditions, human attention is often shaped by explicit task goals. In this work, we address task-driven saliency prediction by proposing a model that conditions visual attention on natural-language task descriptions. The model produces task-dependent saliency maps that reflect how attention shifts under different viewing intents. Through quantitative and qualitative analysis, we show that incorporating explicit task semantics enables more faithful modeling of goal-directed visual attention.

- **核心问题**：现有视觉显著性模型大多假设“自由观看”（free-viewing）条件，即观察者没有明确目标，注意力仅由图像的低层特征（对比度、颜色、方向等）驱动。然而在真实场景（如自动驾驶、人机交互、辅助视觉）中，注意力往往是目标导向的，任务需求会显著改变观察者的注视位置。目前缺乏能够显式地将自然语言任务描述融入显著性生成过程的轻量级模型。
- **研究动机**：填补现有方法在两方面不足：① 未显式利用高层任务语义来指导显著性；② 缺乏将语义线索与空间视觉特征结合的轻量化机制。现有基于Transformer或扩散的复杂架构（如VST、TDiffSal）存在空间分辨率下降或计算成本高的问题。

## 2. 论文提出的方法论：核心思想、关键技术细节

- **核心思想**：显式地将文本任务定义（如“寻找行人”“判断情绪”）作为条件输入，与图像视觉特征通过Transformer融合，使模型能够根据任务意图调整注意力分布。
- **架构组成（四个阶段）**：
  1. **YOLO骨干网络**：使用预训练的YOLOv5su的前10层（直到SPPF模块），输入384×384图像，输出[512,12,12]的特征图（步长32），提供语义丰富且空间精确的视觉特征。
  2. **特征投影模块（FPM）**：单个1×1卷积，将通道数从512降至128，减少计算开销并保留关键空间细节。
  3. **任务编码器**：使用MiniLM-L6 Sentence-BERT将任务文本编码为384维嵌入，再通过线性投影生成128维任务token，与视觉token维度一致。
  4. **Transformer融合模块（TFM）**：单层Transformer编码器，隐藏维度128，4个注意力头。将视觉特征图展开为token序列，并在末尾追加任务token，通过自注意力实现跨模态交互。融合后的序列再重塑为空间特征图。
- **解码器**：逐步上采样和卷积，将12×12特征图恢复至96×96，经Sigmoid输出最终显著性图。训练时降采样至48×48以匹配真值。
- **损失函数**：组合损失 \( L = \alpha \cdot KL(pred \| gt) + \beta \cdot (1 - CC(pred, gt)) \)，其中 \(\alpha = \beta = 1\)，KL散度衡量分布差异，皮尔逊相关系数衡量空间相关性。
- **训练配置**：Adam优化器，固定学习率1e-4，无权重衰减，batch size 8，训练50个epoch。
